Weather-App

To enhance my proficiency in web development and React, I focused on a project to design a weather data application. Initially, I leveraged the OpenWeatherMap API to access global weather data for cities around the world. Simultaneously, I accessed data from GeoNames , which supplied a comprehensive text file containing the names of cities worldwide with populations of at least 15,000. From here I parsed the file to extract pertinent data and constructed the list of cities that users can select for weather information. Given the vast amount of initial data, I implemented a filtering mechanism, resulting in the creation of a refined city list. This curated list exclusively included cities with populations of at least 85,000, optimizing the user experience. Furthermore, I employed virtualization techniques to enhance the application's loading speed, ensuring swift and seamless interactions. Throughout the development process, I utilized conditional rendering to mitigate unnecessary resource consumption. Finally, I deployed the application using Firebase, making it accessible to anyone here. This project served as a valuable opportunity to expand my skill set, encompassing API integration, data parsing, performance optimization, and deployment, culminating in a functional and efficient weather data application.

Pente

Pente is an abstract strategy game for two players, bearing many similarities to the popular game Go. In Pente, players take turns placing stones onto a grid, aiming to achieve either five stones in a row or capture five pairs of their opponents' stones for victory. For this project, I initially implemented Pente in Java as a text-user-interface. It served as an excellent starting point to build my foundational Java skills while providing a project to showcase. As I pushed further into my programming journey, I embraced React and saw an opportunity to take my Pente project to the next level. I recreated the entire game as a graphical-user-interface through React JSX, elevating the game's presentation and accessibility. Moreover, I enhanced the game's playability by adding restart and undo functionalities. These features allow players to modify the game state by resetting the board or undoing their moves, contributing to a more engaging and enjoyable experience. Most recently I utilized Firebase to host the app, allowing anyone to play it here. Pente has been an invaluable project in bolstering my programming abilities, and serves as a testament to my progress in recent months.

Racket Parser & Evaluator

This project was a part of Formal Languages and Functional Programming (CS301) and aimed to create a parser and evaluator working together to evaluate expressions in the procedure-first format utilized by Racket.
  • Initially, I developed an evaluator capable of handling various operations, from simple ones like addition, subtraction, multiplication, and division, to more complex ones like conditionals, mapping, and even lambdas.
  • Following the evaluator's creation, I created a parser that could parse strings using a context-free grammar to properly format the equations.
  • The parser provided the formatted equations to the evaluator which would then map the characters to functions and process them to reach a final answer.
To explore the entire code and delve into the project's specifics, you can visit my Github repository where I have uploaded everything, including a test file with example inputs and their corresponding expected outputs.
