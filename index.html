Racket Parser & Evaluator

This project was a part of Formal Languages and Functional Programming (CS301). The goal was to create a parser and an evaluator that could work in tandem to evaluate expressions in the procedure first format utilized by React. First, I had to create a parser that could parse strings using a context-free-grammar. From this point I created an evaluator that could take the formatted equations from parser, and get the answer. The evaluator has the capacity of handling simple operations such as addition, subtraction, multiplication and division as well as more complex operations such as conditionals, mapping, and even lambdas. To see the entirety of my code and details of the project, check out my Github repository where I've posted everything, including a test file with example inputs and expected outputs.
